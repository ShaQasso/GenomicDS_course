  1. Normalize the counts using the TPM approach.
geneLengths <- counts[,"width"]

counts <- counts %>%
        select(-width)
        
rpk <- apply(counts, 2, 
              function(x) as.numeric(x)/(geneLengths/1000))
tpm <- apply(rpk, 2, function(x) as.numeric(x) / sum(as.numeric(x)) * 10^6)
rownames(tpm) = rownames(counts)

  1. Re-do the heatmaps setting the scale argument to none, and column. Compare the results with scale = ‘row’.

Scaling is important! scaling by row means we scale the counts for each gene separately, so we can compare the specific gene’s expression levels between the samples. Scaling by columns means we scale the counts for each sample separately, which is not very informative as in each sample the genes are expressed in different scales, some having much more transcripts than others.

selectedGenes_V <- names(V[order(V, decreasing = TRUE)][1:500])

heatmap_no_scale <- pheatmap(tpm[selectedGenes_V,], 
                        scale = 'none', 
                        main = "No scaling",
                        silent = TRUE,
                        show_rownames = FALSE)

heatmap_col_sacle <- pheatmap(tpm[selectedGenes_V,], 
                        scale = 'column',
